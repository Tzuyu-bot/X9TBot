# XiaoYaoBot 🗡️🌌

**逍遥九重天**手游自动化脚本，使用 Airtest 实现自动登录、自动打怪、自动领奖等功能，助你轻松修仙，纵横九重天！

---

## ✨ 功能特色

- 🔐 自动登录游戏，快速进入角色界面
- ⚔️ 自动打怪刷副本，支持连续挂机
- 🎁 自动识别并领取奖励
- 🔁 支持循环运行
- 🧠 基于图像识别 + 控件模拟点击，兼容性强

---

## 🛠 技术栈

- [Airtest](https://airtest.netease.com/) – UI 自动化测试框架
- Python 3.x
- ADB + 图像模板识别
