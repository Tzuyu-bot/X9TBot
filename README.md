# X9TBot 

**逍遥九重天**

手游自动化脚本，使用 Airtest 实现自动打怪、自动领奖等功能，助你轻松修仙，纵横九重天！


---

## ✨ 功能特色

- ⚔️ 自动打怪刷副本，支持连续挂机
- 🎁 自动识别并领取奖励
- 🔁 支持循环运行
- 🧠 基于图像识别 + 控件模拟点击，兼容性强

---
